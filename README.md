# Deploy

Netlify: https://agent-6a348df30e7ad89e10d808fc--dashboard-beon.netlify.app/

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
